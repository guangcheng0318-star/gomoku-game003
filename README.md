# 🎮 即時五子棋對戰

一個基於 GitHub Pages 的即時五子棋對戰遊戲，支援自訂棋盤大小和連線長度。

## 🎯 遊戲規則

- 兩位玩家輪流下棋（○ 和 ✕）
- 下滿整個棋盤後計算分數
- 連成指定長度（預設5子）算1條連線
- 水平、垂直、對角線連線都計算
- 連線數量多者獲勝

## 🚀 快速開始

1. **創建房間**：點擊「創建房間」獲得房間ID
2. **分享房間**：把房間ID告訴朋友
3. **加入房間**：朋友輸入房間ID點擊「加入房間」
4. **開始遊戲**：輪流下棋，即時同步！

## ⚙️ 自訂設定

- **棋盤大小**：8x8, 10x10, 12x12, 15x15
- **連線長度**：3子, 4子, 5子, 6子
